What is FluentSecurity ?

FluentSecurity provides a fluent interface for configuring security in ASP.NET MVC and Web API.
No attributes or nasty xml.

Why FluentSecurity ?

The alternatives made me sad

Attributes

  • Dirty
  • Hard to test

XML

  • Magic strings
  • Hard to test

Custom

  • Time consuming

A better way!?

Code based configuration

Refactorable

Quick overview

An example


SecurityConfigurator.Configure<MvcConfiguration>(configuration =>
{
	configuration.GetAuthenticationStatusFrom(() =>
		HttpContext.Current.User.Identity.IsAuthenticated);

	configuration.For<HomeController>().Ignore();
	configuration.For<AccountController>().DenyAuthenticatedAccess();
	configuration.For<AccountController>(x => x.LogOff()).DenyAnonymousAccess();
});
